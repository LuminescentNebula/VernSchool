1) Вращающийся квадрат
   ![alt text](../Примеры/Анимация/loading_1.gif)
    - Создать элемент `square`, указать ему размеры и цвет заднего фона
    - Поскольку здесь две анимации, которые запускаются одна после другой, то нужно для `square` создать контейнер `container`
    ```html
        <body>
            <div class="container">
                <div class="square"></div>
            </div>
        </body>
    ```
    - Для `container` укажим свойство transition с следующеми параметрами:
      - property: transform
      - duration: 1s
      - timing-function: ease
      - delay: 1s
    - Для `square` укажим то же самое, но delay (задержка) установить на 0
    - Задержка в `container` равна времени исполнения анимации в  `square`, поэтому вторая анимация запустится сразу после первой.
    ```html
        <style>
            .container {
                width: 150px;
                height: 150px;
    
                transition-property: transform;
                transition-duration: 1s;
                transition-timing-function: ease;
                transition-delay: 1s;
            }
            .square {
                width: 150px;
                height: 150px;
                background-color: red;
                /*Сокращенная запись - шортхэнд*/
                transition: transform 1s ease 0s;
            }
        </style>
    ```
    - Добавим стили для `container` при наведении мыши с помощью псевднокласса `:hover`. В свойства запишим transform для поворот по оси X на 180 градусов.
    - С помощью комбинаторов добавим стили для `square`, который находится внутри `container`, на который навели мышью. В свойства запишим трансформ по оси Y на 180 градусов
    ```html
        <style>
        /*Это не ещё один элемент style, а тот же самый*/
        /*Создавать их несколько не имеет смысла*/
        .container:hover {
            /*Здесь запишите свойства*/
        }
        /*Комбинатор тоже сами напишите*/
        </style>
    ```
    - Дополнительное задание: скруглить углы квадрата
